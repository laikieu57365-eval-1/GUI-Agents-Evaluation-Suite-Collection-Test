## About ScreenSuite

ScreenSuite is the most comprehensive evaluation suite for GUI Agents created by HuggingFace. It includes 13 benchmarks spanning the full range of GUI agent capabilities:

- **Perception / Grounding**: Correctly perceiving information displayed on screen and understanding the positioning of elements
- **Single-Step Actions**: Solving instructions correctly over one action
- **Multi-Step Agents**: Solving higher-level goals through several actions in a GUI environment

The desktop environment Grounding benchmarks evaluate the ability of GUI agents to correctly identify and locate UI elements in desktop environments (Windows, Ubuntu, etc.).
