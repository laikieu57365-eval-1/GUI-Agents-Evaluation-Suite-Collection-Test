# GUI Agents Evaluation Suite Collection

A collection of information about the HuggingFace GUI Agents Evaluation Suite.

## Desktop Environment Grounding Benchmarks

The following are the desktop environment-related Grounding benchmarks part of the HuggingFace GUI Agents Evaluation Suite:

- OSWorld-G: https://huggingface.co/datasets/MMInstruction/OSWorld-G
- OSWorld Ubuntu: https://huggingface.co/datasets/xlangai/ubuntu_osworld
- OSWorld Windows: https://huggingface.co/datasets/xlangai/windows_osworld
- OSWorld macOS: https://huggingface.co/datasets/xlangai/macos_osworld
